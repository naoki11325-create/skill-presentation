# 商談台本の模範例

相互送客の提携を持ちかける、商談で口頭で話す台本の例。結論（お願い）を先に置き、想定問答で相手の不安を切り返す。口調は「丁寧だが熱意のある口語」。

## 目次

- 対話の流れ（例）
- 例0：Before / After（上流を埋める効果）
- 相手の取り組みを立てる言い換え（NG / OK）
- 例1：塾に持ちかける（主軸）
- 例2：歯科医に持ちかける（表現一段慎重）
- 例3：市役所に持ちかける（後援・共催フレーム）

## 対話の流れ（例）

ユーザーが雑に投げ、AIが1問ずつ推奨案つきで材料を引き出す。まず業種を確定するのが起点。

> ユーザー：スクールの集客を塾に手伝ってもらいたい
> AI：①相手は塾ですね。塾の一番の得は「起立性調節障害や不登校で伸び悩む生徒に打ち手を出せる＝退塾防止・保護者満足」だと想定します。これでいいですか？
> ユーザー：それでOK
> AI：③塾の不安は「怪しい健康商法に見えないか／保護者クレーム／手間」が定番です。想定問答でここに答えます。他に気になる反論はありますか？
> ユーザー：手間を嫌がりそう
> AI：④こちらは柔整院の実在患者を塾に紹介できますね。まず告知だけの軽い一歩＋体験会から、でリスクを下げます。この方向で台本を作ります。

材料が揃うと、下の「例1」のような台本が出力される。

## 例0：Before / After（上流を埋める効果）

同じ「塾に集客協力を頼む」でも、上流（相手の得・不安・こちらの弾）を埋めるかどうかで結果が変わる。

**Before（上流が空っぽ・商品説明になっている）**

```
このたび自律神経の専門スクールを開講します。
自律神経を整えることは健康の土台であり、集中力にも良い影響があります。
つきましては、貴塾でも生徒さんにご案内いただけないでしょうか。
```

自分の都合しか言っておらず、塾に何の得があるか・怪しくないか・手間はどうかに答えていない。塾は「うちに関係ある？」で終わる。

**After（上流を埋めてから話す）**

```
【相手】学習塾の塾長
【お願い（結論）】お互いの生徒・患者を紹介し合う提携をお願いしたいです。まずは告知の協力から、一度お試しで組ませてください。

■ つかみ
先生、勉強はできるのに朝起きられない、集中が続かない生徒さんっていませんか。ああいう子、実は自律神経の不調が背景のことが多いんです。

■ なぜ声をかけたか
先生の指導で伸びる生徒がいる一方、体調やメンタルが壁で伸び悩む子もいると思います。そこは私の領域で、私の院に来る学習ニーズのあるお子さんは先生にご紹介します。お互いの生徒が幸せになる提携です。

■ お願いしたいこと
まずは待合や保護者向けの案内で、うちのスクールを告知していただけないかと。生徒さんに無理に勧めていただく必要はありません。

■ 私が先に出せること
私の院の実在の患者さんで塾を探しているご家庭を、先にご紹介します。あと、まず体験会を1回、貴塾の生徒さん向けに無料でやらせてください。合わなければそこまでで大丈夫です。
```

## 相手の取り組みを立てる言い換え（NG / OK）

「組めばもっと良くなる」は「今のやり方は不十分」に聞こえやすい。相手の取り組みを立て、こちらから先に差し出す。

**NG（相手を否定）**

```
今の塾のやり方だけでは、体調が原因の生徒は救えていないですよね。
```

**OK（相手を立てる＋先に差し出す）**

```
先生の指導で伸びる生徒はたくさんいると思います。
そのうえで、体調が壁になっている子だけ、私が補えたらと思っています。
まずは私の方から、塾を探している患者さんをご紹介させてください。
```

## 例1：塾に持ちかける（主軸）

```
【相手】学習塾の塾長
【お願い（結論）】生徒と患者をお互いに紹介し合う提携をお願いしたいです。まず告知の協力と体験会から、お試しで始めさせてください。

■ つかみ
先生、成績以前に「朝起きられない」「集中が続かない」「なんとなく不調」で伸び悩む生徒さんっていませんか。ああいうお子さん、自律神経の乱れが背景のことが多いんです。

■ なぜ声をかけたか
先生の指導があってこそ生徒は伸びます。ただ、体調やメンタルが壁の子は、勉強を頑張らせるほど空回りしがちで、退塾にもつながりやすい。そこを私が補えれば、先生は指導に集中でき、保護者からの信頼も上がります。うちに来る学習ニーズのあるお子さんは、先生にご紹介します。

■ お願いしたいこと
まずは待合や保護者向けの案内で、うちのスクールを告知していただくだけで大丈夫です。無理に生徒さんへ勧めていただく必要はありません。

■ 私が先に出せること
私の院に来ている、塾を探しているご家庭を先にご紹介します。あわせて、貴塾の生徒さん向けに体験会を1回無料でやらせてください。合わなければそこまでで構いません。

━━ 想定問答 ━━
Q. 怪しい健康商法に見えないか、保護者が不安がりそう
A. 「治る」とは一切言いません。生活リズムや呼吸などのセルフケアを学ぶ場です。保護者向けにも同じ説明をしますし、まず先生に体験会を見ていただいて判断してもらえれば。

Q. 正直、手間をかけたくない
A. お願いは告知だけで、生徒さんへの声かけは不要です。紹介も「合いそうな子がいたら」で構いません。手間はこちらが持ちます。

Q. 本当に成績や出席に効くのか
A. 成績を保証はできません。ただ、朝起きられて体調が整えば、勉強に向かえる土台はできます。まず体調が壁の子だけ、試しに一人紹介いただくのはどうでしょう。
```

## 例2：歯科医に持ちかける（表現一段慎重・金銭なし）

```
【相手】歯科医院の院長
【お願い（結論）】食いしばりや自律神経でお困りの患者さんを、お互いに紹介し合えないかと思っています。まずは情報交換から。

■ つかみ
先生、食いしばりや顎関節の症状で、マウスピースだけでは今ひとつ、という患者さんはいらっしゃいませんか。ああいう方は、自律神経の緊張が背景にあることが多いんです。

■ なぜ声をかけたか
先生の治療があってこその話で、私はその隙間を補う立場です。食いしばりの背景ケアを学べる場をご案内できれば、患者さんの満足につながり、先生の診療の幅も広がると思います。逆に、私の院で歯科が必要な患者さんは先生にご紹介します。

■ お願いしたいこと
まずは、食いしばり・自律神経系でお困りの患者さんに、うちのスクールという選択肢があることを知っていただくだけで十分です。

■ 私が先に出せること
私の院で、歯科の受診が必要な患者さんを先生にご紹介します。まずはお互いの患者像をすり合わせる情報交換からでも。

━━ 想定問答 ━━
Q. うちが紹介して、効果がなかったら信頼に関わる
A. 効果を断定するものではなく、セルフケアを学ぶ場という位置づけです。「治療」ではないと明確に伝えますし、まず先生ご自身に中身を見ていただいてからで結構です。

Q. 紹介料のような話なら受けられない
A. 金銭のやり取りは考えていません。あくまで患者さん同士を紹介し合う、対等な提携としてお願いしたいんです。
```

## 例3：市役所に持ちかける（後援・共催フレーム／相互送客なし）

```
【相手】市役所の健康福祉担当
【お願い（結論）】市民向けの自律神経セルフケア講座を、市の後援または共催という形で一緒にできないかと考えています。

■ つかみ
ご担当者様、最近、睡眠の質や不調の相談は住民の方からも多いのではないでしょうか。自律神経のセルフケアは、そうした未病の入口に手を打てるテーマです。

■ なぜ声をかけたか
特定の事業者を推していただく話ではありません。市民の健康増進・啓発という公共の目的で、学びの機会を一緒に提供したいという趣旨です。

■ お願いしたいこと
市民向け講座を開くにあたり、市の後援名義、あるいは共催という形でご協力いただけないかと。会場のご案内や広報でお力添えいただけると、住民の方にも安心して届きます。

■ 私が先に出せること
講座の企画・講師はこちらで用意します。営利目的の勧誘はせず、あくまで健康啓発の内容に徹します。まずは1回の試験開催から、反応を見ていただければ。

━━ 想定問答 ━━
Q. 特定の民間事業者に肩入れはできない
A. その通りだと思います。だからこそ物販や勧誘はせず、健康啓発の公益イベントとして企画します。中立性を保てる形は、ご担当のご意向に合わせます。

Q. 効果や安全性の裏付けはあるのか
A. 医療行為ではなく、生活習慣・セルフケアを学ぶ内容です。「治る」といった表現は使いません。内容は事前に全て確認いただけます。
```
